## "Arch BTW"
- Have been daily driving arch linux for the last couple of years.
- Arch linux is easily the best linux one can install.
- Everything is setup and configured by me from scratch, giving me full control over bloat, resources and security.
- Its a super fast and efficient workflow, using "hyprland".
- Neovim has been configured nd setup with mre than enough to beat generic IDEs.
- FASTER workflow thanks to vim bindings acorss system (Qutebrowser, thunar, browser extensions, neovim etc..) 
- Refined and finetuned with many sleepless nights of trial and error
